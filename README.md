# usb_modeswitch_Android_binary

Prebuilt binary for [Android port of usb_modeswitch](https://github.com/joerg65/usb_modeswitch_Android)

Built with NDK 21e(21.4.7075529) and libusb 1.0.29, Offering armeabi-v7a and arm64-v8a binary

Tested on Tanix X4(android 11 armeabi-v7a, amlogic s905x4)

Install the executable of your arch to the folder /system/bin or to /vendor/bin on your device.

Install the library to the folder /system/lib (armeabi-v7a) or /system/lib64 (armeabi-v7a) on your device.
